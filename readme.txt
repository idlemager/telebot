📘 产品说明文档（Product Spec）
产品名称（暂定）

TrendPulse
AI 消息驱动型行情雷达 & 风险过滤机器人，可能的话用上trendrader等工具

1. 产品一句话定义

TrendPulse 是一个：

基于实时消息、市场异动和量化风控模型
自动发现“潜在起飞机会”
并以 Telegram 机器人形式
向交易者推送信号与风险建议的 AI 产品。

2. 产品定位
2.1 表面定位（用户感知）

AI 扫描全网消息

提前发现热点币种

自动推送“可能要起飞”的机会

给出建议方向、杠杆、仓位

提供“成功概率感”和风险等级

2.2 实际定位（你真正做的）

一个 垃圾信号过滤器

一个 负期望行为抑制器

一个 爆仓组合压制器

一个 概率意义上更不烂的信号源

你不是预测市场，你是在：

利用人性追热点
同时用量化模型
保住信号长期不至于全是屎。

5. 功能模块拆解
5.1 消息扫描引擎（Trend Scanner）
数据源（v1）

Twitter / X

Telegram 频道

Crypto 新闻 RSS

Reddit

公告类网站（Binance / OKX）

功能

抓取文本

NLP 提取币种关键词

情绪分析（利多 / 利空）

提及量统计

KOL 权重加权

输出指标

消息热度指数（0–100）

情绪方向（多 / 空）

情绪强度

提及量增速

异常程度（z-score）

5.2 市场异动引擎（Market Anomaly Engine）
输入数据

价格

成交量

未平仓量（OI）

Funding Rate

多空比

交易所资金净流入

触发条件（v1）

成交量放大 > 2.5x

价格突破 15m 高点 / 低点

OI 激增 > 15%

Funding 突然翻转

大单扫盘检测

输出指标

价格异动指数

成交量异动指数

杠杆拥挤度

方向一致性

市场状态（趋势 / 震荡）

5.3 叙事匹配引擎（Narrative Engine）
功能

给币种打标签：

ETF

AI

L2

Meme

DeFi

GameFi

RWA

Infra

输出

当前叙事

叙事热度

叙事增长率

叙事生命周期阶段

5.4 风险过滤引擎（Hidden Core）

⚠️ 永远不直接给用户看完整数值

输入

历史价格序列

当前波动率

当前杠杆

Funding 极值

市场结构

当前方向

内部模型

Monte Carlo 路径模拟

历史重放回测

连续亏损概率

最大回撤分布

爆仓概率估计

输出给 UI 层的“壳指标”

风险等级：低 / 中 / 高

建议最大杠杆：2x / 5x / 10x

建议仓位：5% / 10% / 20%

推荐方向：多 / 空 / 观望

6. 信号生成逻辑（v1）
IF
  消息热度指数 > 70
  AND 成交量异动指数 > 60
  AND 方向一致性 > 0.6
  AND 爆仓概率 < 35%
THEN
  生成信号
